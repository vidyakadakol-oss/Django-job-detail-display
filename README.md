Django Job Detail Display

Description

This project is a simple Django web application that shows a list of job postings.
When a user clicks on a job title, the application opens a page that displays the full job details.

Features

- View job listings
- Click a job title to see full details
- Job title, company name, location, and description are displayed
- Data is retrieved using Django ORM

Technologies Used

- Python
- Django
- HTML
- SQLite

How to Run

Run the following commands:

python manage.py migrate
python manage.py runserver

Then open:

http://127.0.0.1:8000/
